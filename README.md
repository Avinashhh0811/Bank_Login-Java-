# Bank_Login-Java-<br><br>
Bank Login Web Application : Java Servlets • JDBC • MySQL • HTML A simple and beginner-friendly banking web application built using Java Servlets, JDBC, and MySQL. The project allows users to register, log in, and check their account balance securely.

Features :
✔ User Registration
✔ User Login (Authentication)
✔ Check Account Balance
✔ MySQL Database Connectivity (JDBC)
✔ Servlet-based Backend Architecture
