# Bank_Login-Java-<br><br>
Bank Login Web Application : Java Servlets • JDBC • MySQL • HTML A simple and beginner-friendly banking web application built using Java Servlets, JDBC, and MySQL. The project allows users to register, log in, and check their account balance securely.

Features :
✔ User Registration
✔ User Login (Authentication)
✔ Check Account Balance
✔ MySQL Database Connectivity (JDBC)
✔ Servlet-based Backend Architecture
<br><br> **Steps to execute :** <br><br>
1)  create a dynamic web project -> add mysql connerctor jar file to project (right click on project -> build path -> configur build path -> go to libraries -> click on class path -> click on add external jars -> create package in src/main/java folder and create three servlet files(right click on src create servlet files)).<br><br>
2) create two html files(right click on webapp folder) in webapp folder (Located in src folder).
